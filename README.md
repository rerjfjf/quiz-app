# 🎮 ЛитКвиз — инструкция по запуску

## Быстрый старт

```bash
# 1. Установить зависимости (один раз)
npm install

# 2. Запустить сервер
node server.js
```

В консоли появится:
```
🎮 LitQuiz запущен!
   Локально:  http://localhost:3000
   По сети:   http://192.168.1.XX:3000      ← эту ссылку даёшь игрокам
   Админка:   http://192.168.1.XX:3000/admin.html
```

> ⚠️ **Все устройства должны быть в одной Wi-Fi сети!**

---

## Смена пароля администратора

В файле `server.js` найди строку:
```js
if (password !== 'admin123')
```
Замени `admin123` на свой пароль. Встречается 4 раза.

---

## Добавление новых авторов / вопросов

Просто создай файл `questions/автор.json`:

```json
{
  "author": "Имя Автора",
  "work": "Название произведения",
  "questions": [
    {
      "q": "Текст вопроса?",
      "options": ["Вариант А", "Вариант Б", "Вариант В", "Вариант Г"],
      "answer": 0
    }
  ]
}
```

- `answer` — индекс правильного ответа (0, 1, 2 или 3)
- Нужно минимум 3 вопроса на автора (из них выбирается 3 случайных)
- Рекомендуется 30 вопросов — тогда каждый раз квиз разный

Или загрузи файл прямо из **админки** через drag-and-drop.

---

## Как проводить квиз

1. Запусти `node server.js`
2. Открой админку: `http://localhost:3000/admin.html`
3. Раздай игрокам ссылку: `http://192.168.X.X:3000`
4. Игроки вводят имя и ждут в лобби
5. В админке нажми **«Начать игру»**
6. Пойдёт обратный отсчёт 5 секунд → вопросы

---

## Механика баллов

- Правильный ответ: **500 очков** + бонус за скорость (до 500)
- Неправильный ответ: **0 очков**
- Максимум за один вопрос: **1000 очков**
